# Phi Placement

Данная программа реализует стандартный алгоритм поиска узлов для вставки Phi функций
на графе потока управления

## Алгоритм действий
1. Читает CFG и Defs
2. Вычисляет множества доминаторов DOM
3. Строит immediate dominators (idom)
4. Строит dominance frontier (DF)
5. Находит узлы для вставки phi-функций

## Структура проекта
- `src/main.cpp` — чтение входа и запуск
- `src/graph.cpp` — парсинг графа
- `src/ssa.cpp` — алгоритм DOM / idom / DF / phi
- `include/graph.hpp`
- `include/ssa.hpp`

## Сборка
```bash
mkdir build
cd build
cmake ..
make
```

## Запуск
./phi_place

## Формат входа
Две строки.

Первая строка — CFG:
{start, 1}, {1, 2}, {1, 3}, ...

Вторая строка — Defs:
{2, 3, 4}

## Формат выхода
Список вершин в фигурных скобках, в которые нужно вставить phi функции.

## Пример

Вход:
CFG: {start, 1}, {1, 2}, {1, 3}, {2, 4}, {3, 4}, {3, 5}, {4, 1}, {4, 5}, {5, stop}
Defs: {2, 3, 4,}

Выход:
{1, 4, 5}
