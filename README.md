# Решения задач для отбора

Здесь находятся решения двух задач для отбора.

- **5.1** — построение дерева циклов на ориентированном графе
- **5.2** — поиск вершин для вставки phi функций на CFG

Оба решения написаны на **C++17**.
Для сборки используется **CMake**.
